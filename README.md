# EDGE Betting Intelligence Platform

AI-powered college basketball betting intelligence. Built for March Madness and beyond.

## Features
- 🔍 AI Game Scout with live web search (injuries, form, ATS trends)
- 📊 KenPom auto-fetch — efficiency ratings pulled automatically  
- 🎲 Monte Carlo simulation — 10,000 outcomes per game
- 8-Metric analysis framework (3PT%, DefEff, Pace, Sharp Money + more)
- 📱 Mobile PWA — installable on iPhone/Android home screen
- 💰 Full bet tracking with P/L dashboard

## Setup
1. Open the app at the GitHub Pages URL
2. Click ⚙ gear → enter your Anthropic API key (get one at console.anthropic.com)
3. Enter a game matchup → hit Run Scout + Simulation
4. Get AI-powered verdict with full reasoning
